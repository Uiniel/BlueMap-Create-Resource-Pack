## Description
This resource pack enables [BlueMap](https://github.com/BlueMap-Minecraft/BlueMap) to render the moving parts of [Create](https://github.com/Creators-of-Create/Create) blocks. To render contraptions as well use this in combination with [BlueMapCreateEntityAddon](https://github.com/BeneHenke/BluemapCreateEntityAddon).

## Installation
Download the zip archive from the [latest release](https://github.com/Uiniel/BlueMap-Create-Resource-Pack/releases/latest) and place it into the packs folder in the BlueMap config directory. Some blocks also require the installation of [BlueMapModelLoaders](https://github.com/Uiniel/BlueMapModelLoaders) (>= version 0.4.2) to be displayed correctly.

## Features
The list of currently supported blocks (more will be added in the future):
- Belts (*)
- Adjustable Chain Gearshift
- Postbox
- Blaze Burner (*)
- Clockwork Bearing
- Clutch
- Cuckoo Clock
- Deployer
- Encased Chain Drive
- Encased Fan
- Fluid Pipes
- Fluid Valve (*)
- Gearboxes
- Gearshift
- Hand Crank
- Bogeys (*)
- Mechanical Arm (*)
- Mechanical Bearing
- Mechanical Drill
- Mechanical Mixer (*)
- Mechanical Press
- Mechanical Pump
- Mechanical Saw
- Millstone
- Speedometer and Stressometer
- Portable Storage Interface
- Water Wheels (*)
- Weighted Ejector
- Windmill Bearing

Blocks marked with (*) require BlueMapModelLoaders to work properly.

The following features are **not** possible with a resource pack and will not work:
- andesite casing on belts (looks like brass)
- water wheel wood type customization
- wooden bracket and metal bracket
- connected fluid tank textures
- rim detection on fluid pipes
